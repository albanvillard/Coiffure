📦 Les 4 Étapes Magiques 🪄

Suivez ce tutoriel simple pour mettre votre site en ligne en moins de 2 minutes.

1

Rassemblez vos fichiers

Assurez-vous que vos fichiers sont bien nommés et organisés dans votre dossier local :

📁 Cours_Coiffure/
  ├── index.html
  ├── biologie.html
  ├── chimie.html
  ├── technologie.html
  ├── quiz.html
  ├── github.html
  ├── 📁 css/
       └── style.css
  └── 📁 js/
       └── main.js
2

Créez un compte GitHub

Rendez-vous sur le site officiel de GitHub (github.com) et inscrivez-vous gratuitement en créant votre compte étudiant ou professionnel.

3

Créez un nouveau projet (Repository)

Créez un nouveau dépôt public nommé cap-coiffure. Cliquez sur le bouton "Upload existing files" et glissez-déposez l'intégralité du dossier Cours_Coiffure (ou sélectionnez tous les fichiers et dossiers intérieurs) directement dans l'interface web pour les téléverser.

4

Activez l'hébergement GitHub Pages

Rendez-vous dans les Settings (paramètres) de votre projet GitHub, puis dans l'onglet vertical Pages. Sous la section "Build and deployment", sélectionnez la branche main (ou master) et le dossier racine / (root), puis cliquez sur Save. Votre site web est maintenant en ligne !

🌟 Pourquoi héberger votre site ?

  • 🚀 Accès Mobile Permanent : Réviser directement sur votre téléphone portable au salon pendant les temps calmes de vos TP.
  • 💼 Book de Recrutement : Envoyer ce lien à des employeurs potentiels de Dijon ou d'ailleurs pour prouver votre sérieux et votre esprit innovant.
  • 🔄 Mises à jour instantanées : Modifiez ou enrichissez vos fiches de cours à mesure de l'année scolaire, le site en ligne se mettra à jour automatiquement.
Votre future URL sera : https://votre-pseudo.github.io/cap-coiffure

💻 Méthode Ligne de Commande (Optionnel)

Si vous utilisez déjà Git en local sur votre machine, voici les commandes pour publier :

# Initialiser le dépôt local

git init

# Ajouter tous les fichiers

git add .

# Valider le commit de départ

git commit -m "Initial commit CAP Coiffure"

# Lier au dépôt distant sur GitHub

git remote add origin https://github.com/pseudo/cap-coiffure.git

# Envoyer les fichiers

git push -u origin main
